Gidole

I spent 4 childhood years in Gidole and therefore named these fonts after it. Gidole is a small beautiful rural town in Ethiopia @ 5°38′57.8″N 37°22′08.7″E overlooking the Great Rift Valley (map).

The Typeface

Design

Gidole is an elegant and highly legible DIN 1451-inspired typeface drawn using Euler spirals. Gidole Play is obviously more playful and therefore better suited for headlines/posters.

Open Source

Everything’s on GitHub - help me by reporting issues or making improvements by forking and creating pull requests. The software I’m using, FontForge, is also free.

License

I’ve made an exception to the Open Font License to allow webfont conversions without the need for renaming. The OFL lets you use and modify the font freely in applications, web, desktop, documents etc.
