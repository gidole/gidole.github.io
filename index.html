Gidole Sans

An open source font by Andreas. View source and contribute on GitHub. You can also download the latest release.

Open Font License

I've made an exception to the OFL to allow webfont conversions without the need for renaming. The OFL lets you use and modify the font freely in applications, web, desktop, documents etc.

Roadmap

» Better spacing + kerning
» Weights 100-900 (I've made a "fake" ligth version)
» Opentype features (alternates, tabular and more)
» Your requests, ideas etc.
