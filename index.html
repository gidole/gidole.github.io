Gidole

I spent 4 childhood years in Gidole and therefore named this font after it. Gidole is a small beautifull rural town in Ethiopia @ 5°38'57.8"N 37°22'08.7"E overlooking the Great Rift Valley (map).

The Typeface

Design

I started creating this somewhat geometric sans serif because I needed a font for an app I'm creating. It's my modern take on the old trusty Deutsches Institut für Normung #1451 (wiki).

Open Source

Everything is on GitHub so you can help me improve this font by reporting issues or making improvements by forking and creating pull requests. I've used the free FontForge outline editor so anyone can contribute. You can also help me improve this webpage.

License

I've made an exception to the Open Font License to allow webfont conversions without the need for renaming. The OFL lets you use and modify the font freely in applications, web, desktop, documents etc.

Usage

I recommend using font size ≥ 14px and you'd probably want to reduce the letterspacing for headlines. For web I'd recommend adding this to your css:
-webkit-font-smoothing: antialiased;
text-rendering: optimizelegibility;
